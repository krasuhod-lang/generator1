ENRICH_LSI / Органичная интеграция LSI-слов в выделенный фрагмент (DSPy-Signature).

[INPUTS]
  selected_text       : HTML/Markdown — фрагмент, в который нужно вписать LSI.
  lsi_unused          : список слов-кандидатов (не покрытых в основной статье).
  niche_terminology   : словарь ниши — для согласования стиля.
  user_instruction    : опциональный акцент пользователя (например: «упор на технику»).

[OUTPUT]
  enriched_fragment   : HTML/Markdown — переписанный selected_text c вписанными LSI.

[REASONING STEPS — выполни молча, в ответе НЕ выводи]
  1) Прочитай selected_text — выясни тему, тон, тип разметки и какие LSI уже стоят.
  2) Из lsi_unused выбери 4–6 кандидатов, СЕМАНТИЧЕСКИ подходящих к теме фрагмента.
     Слова-«пришельцы» (про другую тему) НЕ бери.
  3) Найди в каждом абзаце место для естественной вставки. Если приходится
     перестраивать предложение — перестрой, но СОХРАНИ исходный смысл.
  4) Используй грамматически правильную форму (склоняй по падежу/числу/роду).
     Запрещено вставлять слова в начальной форме «как есть», если это ломает фразу.
  5) Не повторяй один и тот же LSI-термин больше двух раз во фрагменте.

[OUTPUT CONTRACT — обязательные ограничения]
  - Минимум 4 слова из lsi_unused должны попасть в enriched_fragment.
    Если 4 органично НЕ умещается — впиши столько, сколько реально вписывается
    (но НЕ менее 2-х) и оставь остальной текст нетронутым.
  - Слова, которых НЕТ в lsi_unused, НЕ добавляй в качестве «новых ключей».
  - Длина ответа: len(answer) ≤ 1.20 × len(selected_text).
  - Сохрани все исходные H2/H3, абзацы, списки, ссылки, выделения <strong>/<em>.
  - НЕ добавляй вводных конструкций («Стоит отметить…», «В современном мире…»,
    «Важно понимать…»). НЕ начинай абзацы с этих оборотов.
  - НЕ возвращай пустой ответ.

[SELF-AUDIT — пройди молча перед выводом]
  ✓ Сколько LSI-слов из списка вписано? (минимум 2, цель — 4–6)
  ✓ Все ли вписанные слова взяты ИМЕННО из lsi_unused?
  ✓ Сохранена ли SEO-структура (заголовки, абзацы, списки)?
  ✓ Нет ли запрещённых вводных оборотов?
  ✓ Уложился ли в +20 % по длине?
  Если хотя бы один пункт «нет» — переработай ответ ДО вывода.

[STRICT OUTPUT RULES]
  Выведи ТОЛЬКО enriched_fragment — без обёрток ```html```, без префиксов,
  без пояснений. Никогда не возвращай пустой ответ; если ни одного LSI вписать
  невозможно — верни selected_text без изменений (это легитимный исход).
